1. Set name of Service in wmain() function. 
2. Build your project.
3. Run command prompt as administrator.
4. Enter path to ".exe" file.   
5. Enter "install"

	--> Example: "D:\Folder1\Folder2\WindowsService.exe" install

if you need other commands:
	1. Enter path to ".exe" file.
	2. Enter command you need:
		- install
		- start
		- stop
		- restart
		- uninstall

Warnings:
	* If your service is running and you enter "uninstall" command you need stop service to uninstall it.
	* If your service is running you can not run project.
